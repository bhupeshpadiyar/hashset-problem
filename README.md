# hashset-problem
Convert Bean object with integer and list to hash set


# Prerequisite
 * Java 1.8
 * Maven 3.xx

# Steps to run the project

## Step 1: Clone the project 

## Step 2: Go to project folder & run the folloowing commands

```
mvn eclipse:clean
```

```
mvn eclipse:eclipse
```

```
mvn clean compile
```

```
mvn spring-boot:run
```
